# Задача: написать собственный string

<br>**1. Технический стек:**
<br>Язык: C.
<br>Компилятор: gcc.
<br>Прим.: НЕ использовать <string.h>
<br>**2. Структура и сборка:**
<br>Ветка: develop.
<br>Сборка: Makefile.
<br>Тестирование: Для каждой функции написать тесты. Вывод результата: SUCCESS/FAIL в папку test.
<br>**3. Подзадачи:**
<br>s21_strlen — вычисление длины строки.
<br>s21_strcmp — сравнение двух строк.
<br>s21_strcpy — копирование строки.
<br>s21_strcat — конкатенация (добавление) строк.
<br>**4. Как использовать тесты (пример):**
<br>Писать в терминал: 
<br>make rebuild
<br>cd test
<br>./Task_1
<br>./Task_2
<br>./Task_3
<br>./Task_4
<br>make clean
